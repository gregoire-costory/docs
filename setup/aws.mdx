## What you provide

- IAM Role ARN
- S3 bucket name
- CUR 2.0 name

## Prerequisites

- **CUR 2.0** configured
- Parquet recommended
- Hourly or daily export
- Optional: EKS/ECS tags

## Setup steps

1. Use CloudFormation or manual setup
2. Configure CUR 2.0
3. Grant Role access
4. Create AWS datasource in Costory
5. Enter Role ARN + S3 info
6. Request backfill
