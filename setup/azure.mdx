## What you provide

- SAS Token
- Blob SAS URL
- Container name
- Storage account name

## Prerequisites

- Active **Cost Management Export**
- Export frequency: **daily**

## Setup steps

1. Create or locate your export
2. Grant SAS access
3. Create Azure datasource in Costory
4. Enter container + token info
5. Request backfill
